# ARETE Women's Generation

**Purpose. Excellence. Transformation.**

ARETE Women's Generation is a faith-rooted, community-driven organization empowering women to discover purpose, pursue excellence, and create transformative impact. This repository contains the source code for the official website.

**Live Site:** [aretewomengeneration.org](https://aretewomengeneration.org)

---

## About the Organization

ARETE Women's Generation addresses critical gaps in women's empowerment through six strategic programs targeting education, health, economic opportunity, and personal development. Through community-based programs, digital platforms, and transformative experiences, ARETE equips women to thrive personally, spiritually, and socially.

### Programs

- **BLOOM** - Young Mothers Empowerment Program providing vocational skills, educational support, and mentorship
- **PadHer** - Girls' Health & Dignity Initiative promoting menstrual health and school attendance
- **Unfiltered** - YouTube channel sharing authentic stories of purpose, resilience, and transformation
- **Arete Inspirations** - Daily encouragement and empowerment across social media platforms
- **Whole Again Conference** - Annual healing and growth event for women
- **Love in Action** - Community compassion outreach supporting vulnerable families

## Technologies Used

| Technology | Purpose |
|-----------|---------|
| **HTML5** | Semantic markup and page structure |
| **CSS3** | Styling, animations, and responsive design |
| **Vanilla JavaScript** | Interactivity, scroll animations, and lightbox gallery |
| **Google Fonts** | Typography (Cormorant Garamond, Montserrat, Inter) |
| **Vercel** | Hosting, deployment, and SSL |
| **Namecheap** | Domain registration (aretewomengeneration.org) |

### Key Features

- **Fully Responsive** - Optimized for desktop, tablet, and mobile devices
- **Modern UI/UX** - Clean, elegant design with a warm feminine color palette (Dusty Rose, Warm Taupe, Mauve, Cream, Cocoa)
- **Scroll Animations** - Intersection Observer API for fade-in effects on scroll
- **Animated Counters** - Number animations for impact statistics
- **Lightbox Gallery** - Interactive photo gallery from the first Whole Again Conference (September 20, 2025)
- **Sticky Navigation** - Transparent navbar with blur effect on scroll
- **Smooth Scrolling** - Seamless navigation between sections
- **Performance Optimized** - Compressed images, lazy loading, and minimal dependencies
- **Zero Dependencies** - No frameworks or libraries required; pure HTML, CSS, and JavaScript

## Deployment

The site is deployed on **Vercel** with a custom domain from **Namecheap**.

### DNS Configuration

| Type | Host | Value |
|------|------|-------|
| A Record | `@` | `76.76.21.21` |
| CNAME | `www` | `cname.vercel-dns.com` |

### Deploy Locally

To run locally, simply open `index.html` in a browser, or use a local server:

```bash
npx serve . -l 3000
```
